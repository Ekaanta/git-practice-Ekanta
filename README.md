## About This Project
A simple Python project that demonstrates basic programming concepts including:
- Printing output and working with dates
- Basic calculator operations (add, subtract, multiply)
- Function organization and imports
- Git workflow and version control

## Features
- ✅ Display user name and current date
- ✅ Add two numbers
- ✅ Subtract two numbers
- ✅ Multiply two numbers

## How to Use
Run the main program:
```bash
python src/main.py
```
