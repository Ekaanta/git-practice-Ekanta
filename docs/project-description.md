# Python Calculator Project - Detailed Description

## Project Overview
This is a beginner-friendly Python project designed to practice Git version control and basic Python programming concepts. The project implements a simple calculator with basic arithmetic operations.

## Project Objectives
- Learn Git fundamentals (branching, merging, committing)
- Practice Python function definition and imports
- Understand project structure and organization
- Work with version control best practices

## Features
1. **Display Information**
   - User name output
   - Current date display

2. **Calculator Operations**
   - `add(a, b)` - Returns sum of two numbers
   - `subtract(a, b)` - Returns difference of two numbers
   - `multiply(a, b)` - Returns product of two numbers
